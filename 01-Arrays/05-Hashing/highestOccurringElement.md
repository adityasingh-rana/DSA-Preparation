# 📌 Problem: Highest Occurring Element

## 📝 Problem Summary
Find the element with maximum frequency in the array.
If multiple elements have the same frequency, return the smallest one.

---

## 🧠 Approach
- Use hashmap to store frequency of each element
- Traverse array and count occurrences
- Track element with maximum frequency
- In case of tie, return smaller element

---

## ⏱ Complexity
- Time: O(n)
- Space: O(n)

---

## 🧩 Pattern
Hashing / Frequency Count

---

## 🧠 Key Learning
Hashmap allows efficient frequency tracking
Handling tie cases is important for correctness
